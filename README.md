# Simple Interest Calculator

A simple web application to calculate simple interest based on the principal amount, annual interest rate, and time period.

## Formula

The mathematical formula to calculate simple interest is:
$$I = P \times r \times t$$.

Where:
* **$I$** (Interest): Total interest amount
* **$P$** (Principal): The initial principal sum
* **$r$** (Rate): Annual interest rate (in percentage)
* **$t$** (Time): Time period (in years)

## Description

This project provides a straightforward calculator to help users quickly compute the interest earned or paid on a principal sum. It is designed to be lightweight, easy to use, and accurate for basic financial calculations.

## How to use

1. Input the principal amount ($P$).
2. Input the annual rate of interest ($r$).
3. Input the time period in years ($t$).
4. The calculator will output the total simple interest ($I$).
